# JS Our Team 👥

## 📌 Descrizione
Il progetto consiste nella creazione di una pagina web dinamica che visualizza i membri di un team aziendale. Partendo da un array di oggetti, il software genera graficamente delle "Card" per ogni componente, trasformando dati strutturati in elementi visivi.

## 🎯 Obiettivo
Sviluppare un'applicazione che popoli il DOM in modo dinamico e permetta l'interazione dell'utente per l'aggiornamento della lista del team.

### Focus Tecnico:
- **Data Iteration:** Scansione di un array di oggetti contenente informazioni come nome, ruolo e immagine.
- **Dynamic DOM Creation:** Generazione di stringhe HTML (Template Literals) per iniettare dinamicamente le card all'interno di un contenitore nel documento.
- **Layout & Style:** Utilizzo di CSS per organizzare le card in modo ordinato e professionale.
- **Event Handling (Bonus):** Gestione di un form di input per acquisire nuovi membri e aggiornare istantaneamente la visualizzazione della pagina senza ricaricare.
- **Responsiveness (Bonus):** Garantire una corretta visualizzazione su dispositivi mobile e tablet.

## 🛠️ Tecnologie Utilizzate
- **HTML5:** Struttura della pagina e del form di aggiunta.
- **CSS3:** Stilizzazione delle card e gestione del layout responsive.
- **JavaScript (ES6+):** Manipolazione del DOM, gestione degli eventi e dei dati del team.

---

## 🧪 Flusso di Lavoro
1. **Inizializzazione:** Lettura dell'array di oggetti fornito.
2. **Rendering:** Ciclo sull'array per creare il codice HTML di ogni membro e inserimento nel contenitore principale.
3. **Interattività:** Implementazione della logica per raccogliere dati dal form, creare un nuovo oggetto "membro" e aggiungerlo sia all'array che alla pagina.
